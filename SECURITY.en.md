# Security Policy

## Reporting Security Issues

Please report security-sensitive issues to github@aletheion.cn.

Do not disclose sensitive vulnerabilities in public Issues.

## Never Commit

Never commit:

- API keys;
- GitHub tokens;
- Hugging Face tokens;
- OpenAI keys;
- Tencent, Alibaba, Cloudflare, or other cloud credentials;
- email credentials;
- recovery codes;
- private datasets;
- model checkpoints;
- large generated artifacts;
- `.env` files;
- SSH private keys.

## Handling Sensitive Materials

Recruitment questions, machine test materials, interview records, scoring rubrics, candidate evaluations, internal governance, infrastructure handoff materials, and unreleased project plans must be kept in private repositories with limited access.
