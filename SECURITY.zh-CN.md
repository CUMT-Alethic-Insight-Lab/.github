# 安全规则

## 报告安全问题

安全敏感问题请发送至 github@alethicinsight.org。

不要在公开 Issue 中披露敏感漏洞。

## 禁止提交

不要提交：

- API key；
- GitHub token；
- Hugging Face token；
- OpenAI key；
- 腾讯云、阿里云、Cloudflare 或其他云服务凭证；
- 邮箱凭证；
- 恢复码；
- 私有数据集；
- 模型权重；
- 大型生成文件；
- `.env` 文件；
- SSH 私钥。

## 敏感材料处理

真实招新题、机试材料、面试记录、评分标准、候选人评价、内部治理、基础设施交接材料和未公开项目规划必须放在访问受限的私有仓库中。
